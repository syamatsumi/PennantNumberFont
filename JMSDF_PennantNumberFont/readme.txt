This font is paired with shadows.
You will be able to easily place shadowed letters on the texture of the ship.

In order to overlap exactly two glyphs, metrics of the font became a thing very ugly.
Most of the glyphs have the same value for the left sideband.
In other words, it is the specification due to the negligence of the author.
Although it may be troubled with behavior different from ordinary fonts, the author does not know it.

For this font, the width of the low line is set to 1/10 of the cap height in order to adjust the character spacing.
Since lowline is used as a space adjustment space, underlining can not be done with this font.

If you let the retouching software type in it, you can freely make the character spacing a bit.
However, if the character spacing is too narrow, the behavior of "inverse space" will occur.
In other words, the more you insert it, the narrower it is.
The more you use it, the more it will not be understood.
It is better to cope with the software side is basically.

このフォントは影と組になっている。
船のテクスチャに対して容易に影付きの文字入れをすることが出来るだろう。

二つのグリフをぴったり重ねるために、このフォントのメトリクスはとてもブサイクなものになった。
殆どのグリフは左サイドバンドが同一の値になっている。つまり、作者の怠慢による仕様である。
通常のフォントと違う挙動に困るかもしれないが作者は知った事ではない。

このフォントは文字間隔の調整のために、アンダーバーの幅をキャップハイトの1/10に設定してある。
レタッチソフトに字詰めをさせれば文字間隔を若干であるが自由に出来るだろう。
しかし、文字間隔を詰めすぎると「逆スペース」というオモシロ挙動が起きてしまう。
使えば使うほどワケ分からなくなるので急場凌ぎの仕組みと思ってください。
